# Sandbox API

A sample API for experimenting with the JSON:API format. It allows users to create accounts so they can modify their own data.

## Getting Started

### Requirements

1. Ruby
1. PostgreSQL (e.g. [Postgres.app][postgres-app])

### Setup

```sh
$ bundle install
$ rails db:setup
```

### Testing

```sh
$ bin/rspec
```

### Running

```sh
$ rails server
```

[postgres-app]: http://postgresapp.com
